# サンプルSlackメッセージ

このファイルには、ポッドキャスト生成システムの動作確認用のダミーデータが含まれています。

## 使い方

1. 個人のSlackワークスペースでテスト用チャンネルを作成（例：#podcast-test）
2. このファイルの「メッセージ一覧」セクションの内容をコピー
3. テストチャンネルに順番にペースト（メンションは自動的に無効化されます）
4. `.env` ファイルで `SLACK_CHANNEL_SOURCE` にテストチャンネルのIDを設定
5. システムを実行してポッドキャストが生成されることを確認
